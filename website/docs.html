ForgeAI Documentation

Complete reference for the self-hosted AI agent platform. 8 channels, 10 LLM providers, 13 tools, 18 dashboard pages, 7 security modules, 150+ API endpoints.

Quick Start

Get ForgeAI running in under 2 minutes:

git clone https://github.com/forgeai-dev/ForgeAI.git
cd ForgeAI
pnpm install
pnpm forge onboard    # Interactive setup wizard
pnpm forge start

Prerequisites: Node.js ≥ 22, pnpm ≥ 9, MySQL 8.x

The Gateway starts at http://127.0.0.1:18800 with the Dashboard included.

Installation

Docker (Recommended)

docker-compose up -d

This starts the Gateway + MySQL with health checks, volumes, and Chromium for browser tools.

From Source

pnpm install
pnpm build
pnpm forge onboard --install
pnpm forge start --migrate

Configuration

ForgeAI uses .env for initial config and the encrypted Vault for secrets. Run pnpm forge onboard for interactive setup, or manually edit .env.example.

Key environment variables:

VariableDescription
JWT_SECRETSecret for JWT token signing (auto-generated by onboard)
VAULT_MASTER_PASSWORDMaster password for AES-256-GCM encrypted Vault
MYSQL_HOSTMySQL connection host (default: 127.0.0.1)
PORTGateway port (default: 18800)

API keys for LLM providers are stored encrypted in the Vault, not in .env. Configure them via Dashboard → Settings.

Automatic Failover & Circuit Breaker

Every provider has circuit breaker protection (5-failure threshold, 2-minute cooldown), exponential backoff retries, and automatic fallback to the next provider in the chain. If all cloud providers go down, your local model picks up.

Browser Tool (Advanced)

The browser tool supports 25 actions: navigate, screenshot, content, click, type, scroll, hover, select, back/forward/reload, wait, cookies/set_cookie/clear_cookies, extract_table, evaluate, pdf, new_tab/switch_tab/close_tab, upload, switch_profile, list_profiles, snapshot, close.

Multi-Profile: Each profile has its own userDataDir with persistent cookies, logins, and localStorage. Switch profiles on the fly: browser({ action: "switch_profile", profile: "gmail" })

File Upload: Upload files via CSS selector to input[type=file] or automatic file chooser interception.

DOM Snapshots: Capture full page state (cookies, localStorage, sessionStorage, forms, scroll position) as JSON in .forgeai/snapshots/.

Desktop Automation

Cross-platform (Windows/macOS/Linux): mouse control, keyboard typing, screenshots, OCR text extraction, window management, clipboard read/write. macOS via AppleScript, Linux with X11+Wayland support.

Credential Vault

All API keys, tokens, and secrets are encrypted with AES-256-GCM and derived from a master password via PBKDF2 (310,000 iterations). The Vault is file-persistent and never stores secrets in plain text.

DM Pairing

Onboard users securely with invite codes (FORGE-XXXX-XXXX). Generate codes from the Dashboard or API with configurable expiration, max uses, role assignment, and channel restriction. Users redeem codes with /pair from any messaging channel.

Agentic Loop (25 iterations)

The agent autonomously iterates: think → decide tool → execute → process result → repeat. Up to 25 iterations per request. Each iteration is tracked with real-time progress broadcast via WebSocket.

Multi-Agent System

Create multiple agents with different models, providers, personas, and tool permissions. Route messages to specific agents based on channel, peer, or session bindings. Agents communicate with each other via session tools (sessions_list, sessions_history, sessions_send).

RAG Engine

Ingest documents (text, markdown, PDF), chunk them with configurable overlap, generate TF-IDF embeddings, and search semantically. Retrieved context is injected into the agent's prompt automatically. Managed via Dashboard RAG page or API.

Cross-Session Memory

TF-IDF-based memory that persists across sessions. The agent automatically stores important context and injects relevant memories into new conversations. Consolidation removes duplicates and merges related entries.

Voice & Wake Word

Voice Engine: TTS (OpenAI, ElevenLabs, Piper) and STT (Whisper). Full voice chat pipeline: audio in → STT → Agent → TTS → audio out.

Wake Word: Always-listening detection with Porcupine/Picovoice. Say "Hey Forge" to activate the agent hands-free. Configurable sensitivity, keyword selection, and detection stats in Dashboard.

Autopilot

Define scheduled tasks in AUTOPILOT.md with smart tags: @startup, @hourly, @morning, @afternoon, @evening, @daily, @weekly. The engine parses the file and executes tasks automatically. Results delivered via Telegram. Editable from Dashboard Workspace page.

Docker Deployment

docker-compose up -d

The docker-compose.yml includes:

  • Multi-stage Dockerfile (build + production + Chromium)
  • MySQL 8.x with health checks
  • Persistent volumes for Vault, workspace, and sessions
  • Auto-restart on failure

Remote Access

Tailscale: Zero-config remote access without port forwarding. GET /api/remote/status checks Tailscale availability.

IP Filter: Allowlist/blocklist mode with CIDR support and private range detection.

OAuth2/SSO: Google, GitHub, and Microsoft authentication via authorization code flow.
